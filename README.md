## Setup Instructions
1. Run `npm install` to install dependencies.
2. Ensure MongoDB is running on `localhost:27017`.
3. Create a `.env` file with `SESSION_SECRET` and `MONGO_URI`.
4. Run `npm start` to launch the server at `http://localhost:3000`.
